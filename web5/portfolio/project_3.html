Magazine Cover

The magazine cover was for my Publication 2 class. I created a fashion magazine that showcases modest fashion for women. I chose a close-up shot of a modestly dressed fashionable woman. I used black and white font to keep the magazine cover simple yet elegant. I incorporated a bar code and date of the magazine issue to add a realistic element to the magazine cover. I created a name for my magazine which I called “Modestly. Co”.
